# go_trumpcards
トランプカードゲームアルゴリズムをGoで実装

## Description
トランプカードのアルゴリズムをGo+Clean Architectureで実装したプロジェクトです。

## Usage
### Install
```sh
git clone https://github.com/yuta-yoshinaga/go_trumpcards.git
cd go_trumpcards
go run main.go cui
```
